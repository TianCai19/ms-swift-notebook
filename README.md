- **定位**: 个人的 ms-swift 学习与实践笔记
- **状态**: 持续更新

### 快速导航
- **简介**: `notes/00-简介.md`
- **环境与升级**: `notes/10-环境与工具.md`
- **数据与SFT训练**: `notes/20-数据与SFT训练.md`

### 使用方式
- 直接在 `notes/` 目录下新增或编辑 Markdown 笔记
- 命令或代码以 fenced code block 记录，方便复制与复习

### 目录结构（约定）
- `notes/` 存放分主题笔记
- 命名建议采用前缀编号，便于排序（例如 `10-环境与工具.md`、`20-语法基础.md`）
